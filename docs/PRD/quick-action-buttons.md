### Description

Integrate quick action buttons within the Investment Buddy chat interface to allow users to take immediate action on recommendations and suggestions without leaving the conversation.

### Acceptance Criteria

1. Quick action buttons are displayed alongside relevant recommendations (e.g., Apply Strategy, Add to Watchlist, Run Backtest)
2. Clicking a button triggers the corresponding action and provides immediate feedback
3. Actions are executed seamlessly without disrupting the conversation flow
4. Users can undo or modify actions through follow-up messages
5. Button availability is context-aware based on the type of recommendation
